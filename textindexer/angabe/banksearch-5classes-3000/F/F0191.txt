Upcoming Events
 
 
15-May-02
 
Web Services and Microsoft .NET
 
 
 
Presented by Christopher Desany - Keane, Inc.
 The discussion will start with a brief overview of Microsoft's .NET
 initiative, paying particular attention to what web services are
 and what a web services business model may be composed of. A data
 driven web service will then be created in C# using Visual Studio
 .NET, and the subsequent WSDL and SOAP messages analyzed.
 Chris is a Microsoft Certified System Engineer, Database Administrator,
 and Solution Developer. He is a Principal Consultant and web architect
 for Keane, Inc. a $1 billion technology consulting services firm.
Link to
http://www.cs.albany.edu/~desany/CDVBUG/frame.htm
for the presentation
Link to
http://www.cs.albany.edu/~desany/CDVBUG/CDVBUG_WebServices.zip
for the code
To email Chris: desany@hotamil.com 
 
 
20-Feb-02
 
XML with .NET
 
 
 
This month, Justin Whiting of RAD will be giving a presentation introducing the group to
 Extensible Markup Language (XML). He will be discussing and giving demonstrations of the language’s structure and use, specifically in the context of web services, SOAP, UDDI, WSD, and .NET
 
 
16-Jan-02
 
Intro to .NET
 
 
 
This month, Chuck Snead of RAD will be giving a presentation introducing the
group to .Net technology. He will be discussing and giving demonstrations of
the architectural changes between VB6 and VB.Net, memory management, free
threading, and inheritance.
