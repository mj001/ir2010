	An utility to stress your web server
	
	
	
	7/9
	
	
	
	article
	Francesco Balena's WebStresser utility is a simple VB6 application that repeatedly sends an HTTP request to the Web server of your choice. You can use it to test the ASP or ASP.NET application you're working on, or you can run multiple instances of this EXE to see how your app reacts to concurrent requests. The full source code is provided, so you can modify its user interface and internal workings as desired.
