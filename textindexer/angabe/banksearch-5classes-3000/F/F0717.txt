	Visual UML 
	 
	Complete Support for ALL Unified Modeling Language (UML) Diagram Types 
	From Visual Object Modelers 
	 
	
 
Visual UML is an affordable, easy-to-use, yet powerful and full-featured object-modeling tool that supports all OMG 1.3 UML (Unified Modeling Language) diagram types: Use Case, Package, Class, Object, State, Collaboration, Activity, Deployment, Component and Sequence diagrams.
 
Visual UML Developer Edition adds several integrated, bi-directional language interfaces that support the generation and reverse-engineering of Visual Basic, C# and Java code. Plus, generation of C++ code. Plus database schema generation and reverse-engineering of several database types. Visual UML Developer Edition can also run as an add-in to the Visual Basic IDE (for generating and reverse-engineering Visual Basic projects) and the Visual Studio.NET IDE (for generating and reverse-engineering C# projects.)
Visual UML Plus Edition adds integrated support for Visual Basic for Applications (VBA) 6.3.
Visual UML for Visual FoxPro adds integrated round-trip code generation and reverse-engineering of Visual FoxPro projects.

	Which Visual UML is Right for Me?
- Visual UML v2.9.2 Standard Edition is the entry-level product. It implements the Unified Modeling Language (UML) 1.3 through an integrated model/data dictionary and comprehensive support for Class, Object, Package, Use Case, Component, Deployment, Collaboration, Sequence, Activity, and State diagrams.
- Visual UML v2.9.2 Developer includes all the features of the Standard Edition plus it adds several integrated, bi-directional language interfaces that support the generation and reverse-engineering of Visual Basic, C# and Java code. Plus, generation of C++ code. Plus database schema generation and reverse-engineering of several database types. Visual UML Developer Edition can also run as an add-in to the Visual Basic IDE (for generating and reverse-engineering Visual Basic projects) and the Visual Studio.NET IDE (for generating and reverse-engineering C# projects).
- Visual UML v2.9.2 Plus Edition contains the same features as the Standard Edition plus integrated support for Visual Basic for Applications (VBA) 6.3.
- Visual UML v2.9.2 Plus Developer includes all of the features of the Plus Edition plus it adds several integrated, bi-directional language interfaces that support the generation and reverse-engineering of Visual Basic, C# and Java code. Plus, generation of C++ code. Plus database schema generation and reverse-engineering of several database types. Visual UML Developer Edition can also run as an add-in to the Visual Basic IDE (for generating and reverse-engineering Visual Basic projects) and the Visual Studio.NET IDE (for generating and reverse-engineering C# projects).
- Visual UML v2.9.2 Standard for Visual FoxPro contains the same features as the Standard Edition plus a bi-directional interface to Visual FoxPro. This edition supports round-trip code generation and reverse-engineering of Visual FoxPro projects.
- Visual UML v2.9.2 Plus for Visual FoxPro contains the same features as the Plus Edition plus a bi-directional interface to Visual FoxPro. This edition supports round-trip code generation and reverse-engineering of Visual FoxPro projects.
