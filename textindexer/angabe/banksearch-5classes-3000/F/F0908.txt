Code for Registry Class Series 
This code provides the class and a test application to use the Registry class discussed in the article series. File Size: 5 Kb 
 
Part 1: Understanding the Windows Registry 
This part of the article series introduces the concepts used by the Windows Registry to manage the information needed by Windows to function properly. 
 
Part 2: Getting Started 
This part of the article will show you how to build a class that can access all parts of the Registry. The class is completely self-contained and gives you access to all the functions that the API provides. 
 
Part 3: The Registry Class Properties 
This part of the article covers the properties required for the Registry class to work properly. 
 
Part 4: The Registry Class Methods 
This part of the article covers the methods that you need for the Registry class to function properly. 
 
Part 5: Putting the Class to Work 
Now that you've finished the code for the class, you'll learn how to use the class in your own applications.
