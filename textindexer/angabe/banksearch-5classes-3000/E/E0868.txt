Microsoft Visual C++® .NET is the most productive C++ tool for creating the highest-performance applications for Microsoft Windows® and the Web. Nearly all world-class software, ranging from the leading Web browsers to mission-critical corporate applications, is built using the Visual C++ .NET development system. Visual C++ .NET brings a new level of productivity to C++, without compromising flexibility, performance, or control.
