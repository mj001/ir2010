	Visual C++ .NET Product Overview 
	
	
	 
	 
	
	
	 
	
	
	
	
 
 
 
 
 
 
 
Microsoft Visual C++® .NET Standard Edition is available for the hobbyist, student, or beginner programmer looking to learn the Visual C++ 
language by constructing applications for Microsoft Windows® and the Web. 
Professional C++ developers looking to construct complex data-driven or server-based solutions, including XML Web services, should get 
Visual Studio .NET, available in Professional, Enterprise Developer, and Enterprise Architect editions. 
Compare the following features to see which edition best meets your development needs. 
 
 Build Windows-based and .NET-based Applications in C++ 
 
 
 Product/Feature 
 Visual C++ .NET
Standard 
 Visual Studio .NET
Professional 
 
 
 
 Enhanced professional C++ Optimizing Compiler
 Build and deploy professional-quality applications with the industry's leading C++ Optimizing Compiler, which generates the smallest 
 and highest performing executables. Now with Whole Program Optimization technology. 
 
 
 
 
 
 
 Non-optimizing C++ Compiler
 Build and deploy C++ applications with performance and size characteristics appropriate for hobbyist, student, and beginning programmers. 
 
 
 
 
 
 
 New Visual C++ .NET Managed Extensions
 Enable new and existing native Windows-based applications with .NET-based capabilities, such as garbage collection, and .NET-based 
 component technology. 
 
 
 
 
 
 
 Enhanced ANSI/ISO C++ language conformance 
 Visual C++ .NET is Microsoft's most ANSI/ISO C++ conformant compiler ever, with new support for standard-defined features, including 
 covariant return types. 
 
 
 
 
 
 
 Deploy more robust and secure applications
 Visual C++ .NET includes /GS, a new compiler feature that serves the developer as a "backup parachute," catching some of the more 
 common attacks against applications. 
 
 
 
 
 
 
 The .NET Framework and the common language runtime
 Build, deploy, and run distributed XML Web services and Windows-based applications and components on the agile, scalable environment 
 of the .NET Framework. 
 
 
 
 
 
 
 Employ declarative programming with C++ Attributes
 All new C++ Attributes provide developers with the ability to generate robust code more quickly by using Interface Definition Language 
 (IDL)-style attributes directly in C++ source code. 
 
 
 
 
 
 
 ATL Server for XML Web services and Web-based applications
 Active Template Library (ATL) Server classes in the ATL provide developers with high-performance classes that fully encapsulate the 
 best practices for developing Internet Server Application Programming Interface (ISAPI)-based XML Web services and dynamic Web 
 applications in C++. 
 
 
 
 
 
 
 Enhanced MFC 7.0 and ATL 7.0 Libraries
 Leverage the most robust, productive, and popular application framework libraries available for Windows-industry-standard Microsoft 
 Foundation Classes (MFC) and ATL, now enhanced for Windows XP. 
 
 
 
 
 
 
 Enhanced Standard C++ Library
 The C++ Standard Template Library (STL) is also enriched with increased ANSI/ISO standards compliance, new container classes, better 
 support for thread-safe coding, and greatly enhanced source code readability (code formatting). 
 
 
 
 
 
 
 Enhanced Edit and Continue debugging
 Make and apply code changes while debugging, maintain application state, then continue without rebuilding for faster turnaround. 
 
 
 
 
 
 
 Enhanced advanced application wizards
 Easily generate advanced C++ projects with professional-level application wizards for MFC ISAPI dynamic-link libraries (DLLs), Custom 
 Visual Studio Wizards, and more.
