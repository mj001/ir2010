:: WMFWriter2D
The WMFWriter Java package gives you the possibility to write Windows 
Metafiles (WMFs). It works with JDK 1.0, JDK 1.1, JDK 1.2, JDK 1.3 and JDK 
1.4. The WMFGraphics class inherits from the java.awt.Graphics object and 
gives you the chance to create Windows Metafiles by your standard Java 
drawing routines.
http://jars.com/classes/jresout.cgi?resource=10222
:: Jahia Portal Server 3.0
Jahia Ltd. Open source and 100% Java based solution for building complex 
and dynamic Web portals.
http://ipw.internet.com/e-business/intranet/970472986.html
:: jGRASP
jGRASP is a multi-platform, multi-language IDE that adds Control Structure 
Diagrams to Java, C, C++, Objective-C, Ada, and VHDL source code. The 
Control Structure Diagram (CSD) is a control flow and data structure 
diagram that is designed to fit into the space that is normally taken by 
indentation in source code. The intention of the CSD is to improve the 
comprehension efficiency of source code and, as a result, improve software 
reliability and reduce software costs.
http://jars.com/classes/jresout.cgi?resource=10212
:: JImager
JImager is user friendly image viewer written in pure Java. It supports 
file functions like save as, delete, rename and etc., and image viewing 
functions like zoom, set size, fit to desktop, resize image by dragging 
windows, slideshow, etc.
http://jars.com/classes/jresout.cgi?resource=10209
:: Juliet 3
Pioneering Intelligence Augmentation for Java Developers: the free version 
of Juliet enables Java programmers to better and faster understand source 
code and helps Java development teams communicate and document their 
codebase. Juliet is pointed at source code in zips/jars or dirs, and can 
then be used to gain an overview of the codebase (package browser and type 
diagrams), read and browse the code (using a fully hyperlinked source code 
visualizer and browser), and do impact analysis.
http://jars.com/classes/jresout.cgi?resource=10210
