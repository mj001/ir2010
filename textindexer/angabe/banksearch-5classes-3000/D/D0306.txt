Canadian Mortgage
A calculator designed for fixed Canadian Mortgages. It allows
"what if" analysis to determine the best mortgage to suite
your budget.
CADViewer
CADViewer is a Java applet which allows you to view AutoCAD DWF drawings on the Web.
Calc
The Calc applet is an actual calculator with advanced functions including symbolic differentiation.
Calctag
The FREE calculator applet, Calctag, is a handy calculator that can be embedded in a Web page.
calculator
This fast-loading, free applet computes powers, multiplies, divides and summarizes.
calendar
The calendar shows a different image for each month, and users can go backwards or forwards in the calendar year.
CalendarServlet
A free server-side appointment calendar.
CanoeClobber
Crazy Carl loves to fish but he has no fishing pole or net. So, he just runs the river, clobbering his prey. See what you can clobber for supper!
CarGame
CarGame is a free racing car applet developed by author bharath for fun. Source code is included, so you can learn while you play!
Cartoline
Add e-card services to your site with Cartolina Servlet.
CascadeMenu
A compact vertical or horizontal menu interface with multiple level sub menus.
Included is a complete visual real-time editor to guide you through the entire 
set-up process without the need of coding! 
Castles
This Java game is based on an early computer game. For one to four players, or test your skills against the computer! 
Catalog View
Catalog View is a free database applet that can display your data in a grid display.
Country Code
Ever wonder what those .us or .de or .fr stand for in some Web domain names? Now you can find out using this slick applet!
ccalendar
CCalendar is a free calendar applet that you can use as a real calendar, choosing the global appearance and photos you want to show.
centipedo
Rid your garden of pests in this classic video game.
Cerebellum
A 2D menu with a spreading tree structure provides easy navigation while allowing visitors to visualize your site! Looks great, highly configurable, and fun to play with!
CerebellumImage
An image based, 2D menu with a spreading tree structure that provides easy navigation while allowing visitors to visualize your site! Looks great, highly configurable, and fun to play with! HTML code generator included!
CFade
A really cool crossfade between images! CFade comes as part of the Anfy application which not only configures the applet for you, but also generates the HTML code! All ~you~ do is click buttons, no programming experience required! 
CGIFormParser
This class URL decodes and works with a string resulting from a CGI input form.
cgolf
So the boss made you get rid of that carpet golf game you used to play in the office? Time to check out Ed Hobbs' free Java version of the game!
Chart
Plots a bar chart with various options.
ChartMaker
ChartMaker Applet makes pie charts, bar charts, and line charts via
JavaScript.
ChatServlet
ChatServlet is a Java servlet that implements chat rooms. It is a pure HTML implementation, so this chat should be free from firewall restrictions.
Chemis3D
Chemis3D is a free Java Applet for the online 3D Visualization of Molecular Models. Chemis3D uses most of the common chemical datafile formats.
Childs_Paint
A child's paint program that offers all of the usual features like spray, fill etc. The program can be run as an applet or as an application, but typing "java PaintApp" at the command line. 
ChompText
ChompMan eats your text! You can even set the time between feedings!
Works better in IE.
Choose your Calendar
This applet displays the calendar of any month and year combination. Find out the day of the week you were born! It's very easy!
ChronoLisa
The ChronoLisa clock applet was originally written to help promote its real life counterpart, a digital electronic clock with a unique display called the Chrono Lisa. This clock features the time represented in three interlocking ellipses of 60 LED's each. (One ellipse represents hours, the next is minutes, and the third is seconds).
CircleBanner
If you're looking for a professional looking effect for your site, CircleBanner may be just the applet. You can change the parameters for the font, color, direction, size, background image and more on this free applet.
Cliffie
Quite a programming feat, and a real showcase of using sprites, Cliffie is one of the slickest games on the JavaBoutique.
Cline
Cline is another free button applet. The surface of the button is algorimetically animated, and has several moving colored lines on it.
ClipControl
Play or loop an .au file in a web page, with optional controls.
Clock
The original analogue clock applet from Sun Microsystems. Source is included.
Clock2
A clock that can be configured in a variety of ways.
CodeBrain's Slider
CodeBrain's Slider slide-show applet supports up to 100 images, of the
same or differing sizes. Ready-to-run demos and detailed instructions
are provided with the free download of this small, fast applet.
CodeBrainVoice
CodeBrain's VOICE is a very fast, very small, multi-purpose
sound applet.
ColorFader
This little applet passes your image through the full color spectrum.
ColorFinder
This applet lets you use scrollbars to modify the Red, Green and Blue colors in the applet, instantly showing the hexadecimal value of the new color.
Color Picker
Pick your color with this hexadecimal color code generator!
colors
Displays colors for user-defined RGB hexadecimal combinations.
Colormenu
A Colorful menu applet with mouseover effect. Highly configurable.
Comes with commented source code, great for learning Java!
Color Text Scroller
 
A customizable multi-color scrolling text message with mouseover links. Configure this scroller to match your site by adding your own background/foreground images, and specifying the color value of each line of text! 
ColorUp
With a picture that you specify, the ColorUp Applet can animate the color from full color to gray scale or vice versa.
Jakes Coloring Book
This is a free on-line coloring book applet, and would be perfect for children's sites.
Concentration
An old school mental challenge: the classic card game Memory. Use your own images by editing the cards.gif file.
Confetti
Starting life as snow, this trendy applet was converted to confetti by popular
request! Highly configurable, including various amounts of up to 13 different
colors, pixel size, delay, fits any size image and supports frame overlay!
Connect4
Just like you remember from when you were a kid, but the computer opponent is tougher than your friends were! Source included.
Contour
An interesting twist to navigation, this applet can be used with image
buttons or text, both of which are overlayed with a roatating geometric.
ConversionApplet
The free ConversionApplet enables visitors to convert units of measure from one to another. Converts length, Temperature, Area, Volume and Mass.
Convert
This handy little applet will convert any unit to any other unit
with the click of a button!
Convertor1
Convert numbers up to a billion into their text equivelant! 
CoolBanner
This applet takes any number of banners (468x60) and rotates them in order using different cool transitional effects to hide one and display the next. Highly configurable. 
CoolMenu
Create unlimited menu items, without knowing HTML or Java, with this slick GUI application! All required files, (html, image, audio, class), are copied to the directory you specify with the click of a button! Awesome! 
CoolScroll
CoolScroll features smooth scrolling text (using your own fonts!) which bounces up and down and fades in and out!
coowild
If you'd like to provide your visitors with a unique screensaver, the freeware coowild screensaver applet may be just what you're looking for.
Corf_Scroller
This is a small horizontal text scroller applet (7k) with source code. It is unique in that it sets font size automatically. Highly configurable with url launch included.
Cornholio
Let users call up sound files with this basic audio applet. Source is included, and this is a good example if you want to begin coding sound in your Java projects.
countdown
Counts down to a user-defined date and time.
CountDown2
This freeware applet displays the number of days till any specified date. You can use your own digit images, define a background image or color and set the number of pixels between the digits. Source included!
Cross Fader
This little applet produces two lines of text that fade in/out over top of each other. Source included!
crossword
A 15x15 crossword puzzle.
CSGuestbook
CSGuestbook is a free, open source Java servlet that allows web site users to sign and view multiple electronic guestbooks.
CTreeApplet
A configurable navigation tree applet in the style of Windows Explorer that is perfect for creating site maps and formatting links.
CubeApp
Rotate the cube by dragging it with the mouse button; move the pieces by holding "control" and dragging one piece to a neighboring piece with the mouse button. The Rubik's Cube of the Web.
cuppaTOC
cuppaTOC is a folder-based tree menu applet that can display a large number of content lines in a compact browser space without the need for scrollbars.
Cursor
Make some cool 3D effects with this applet. Every time you click a new line is added!
Cybercolors
This cool little applet can determine the Hex Code for any of the 16 million
colors, or for the safety minded a check box will restrict you to the Browser
Safe colors.
