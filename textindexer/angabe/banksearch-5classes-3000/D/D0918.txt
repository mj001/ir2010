VasTabs
VasTab is an excellent tabbed interface for navigation on your site. Customizable colors, button sizes, and more!
VasJava2HTML
This applet makes your Java code more readable by identifying and color-coding keywords with HTML. You can specify your own colors and keywords with parameters.
VbarchartApplet
This 2D / 3D Vertical Bar Graph provides both a client and server side solution for the incorporation of bar graphs into web pages.
vcid
VirtualCID provides you with the ability to view and save callerID information from a callerID enabled modem on your computer.
Venturi Tube Calculator
Venturi tube calculator. Input diameters, measured pressures, fluid density and calculate flow rates, velocities and more. Good illustration of the Bernouli principle. 
Vertical Menu System
A professional two-level, vertical, navigation system. 
VertTextScroller
This applet features smooth scrolling, manual or automatic pausing, highlighted URL links, error checking, and many fonts, styles, and colors to choose from.
VertTicker
Slick Ticker includes URL support, text and background color, bold and italics.
Video Poker
A versatile Video Poker applet. Loaded at run time from html
files, these games are highly configurable. Simply edit the
html file to create different poker games and card designs!
Video Slot Machine
A small, slot machine like game that has the ability to generate 
a unique code for every win.
ViewTree
A nifty little tree hierarchy applet. The evaluation version can be configured for a maximum of 20 branches, 5 images, 4 colors, 4 fonts and unlimited nesting.
VirtualBanner
A virtual city, complete with animated trains and planes. 
Virtual Max Memory Check
Memory Check is a memory game reminiscent of the old TV Game Show 
"Concentration". Lots of customization 
here, including custom images, text color, background color, row and 
column width and height. Includes a time clock and keeps a record
of the number of clicks.
VisualScrollFx
This text scrolling applet loads fast, is smooth, features fades, swipes and many other slick visual effects...and adds some class to your page!
Vivace
Plays .au files quickly each time a page is loaded. Source (only about 7 lines) is included.
VJFE
A simple typewriter applet with easy parameter configuration.
VMaxChat
Virtual Max Chat is a professional-quality chat server/client package.
Vrml
Parses and displays wireframe views of VRML (*.wrl) files.
vrover
This menu applet looks like array of buttons with flashing lamps.
VScroller
VScroller is a highly configurable Vertical scroller. Mix pictures and text, have complete control on layout, even put a date range on items to show or not to show an item! 
VScrollerTypeWriter
The free multi-line, vertical scroller typewriter applet has many options including the ability to automatically wrap text at borders, customize background & foreground colors, font size, font name, etc.
Vertical Scrolling Window
The Vertical Scrolling Window supports a host of features for presenting all types of information.
vText
vText is a compact text and image microbrowser. You can get precise text composing, visualization of long contents is small frames, vertical scrollers (manual or automatic), content navigators, news tickers, banners...
