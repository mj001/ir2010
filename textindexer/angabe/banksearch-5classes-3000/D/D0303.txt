 mailing lists 
 
 
 
	There are two mailing lists available for subscription:
	
	
- japhar@hungry.com
	This list's charter is to provide a forum for all
	announcements and public discussion for or relating to japhar.
	
	To subscribe, send a blank message to
	japhar-subscribe@hungry.com.
	
	
- japhar-commits@hungry.com
	This list receives all CVS commit messages for japhar. If
	you're tracking japhar using anonymous CVS, you should
	consider subscribing to this list.
	
	To subscribe, send a blank message to
	japhar-commits-subscribe@hungry.com.
