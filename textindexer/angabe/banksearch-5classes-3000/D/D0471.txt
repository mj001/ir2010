BallDrop
Tiny balls drop to form a normal, bell-shaped distribution. Source included.
banner
This banner applet cycles through a list of text messages, each with a corresponding link. Source included!
Banner2
The Banner applet displays a series of images in a loop. If an image is clicked, the browser loads a specified url. The applet is free and source is included.
BannerDisplay
Displays images in either random or sequence of list. Each image can link to a
URL with configurable target frame. Images do not need to be the same size. Free.
BannerManPro
Transition between banners, or images of any size, with these awesome special effects! 
Bap
A nice breakout clone with adjustable speeds.
Bar Chart
This applet is designed to take multiple series of numerical data and
display the information as a 3D Multi-series bar chart.
Highly configurable and easy to use.
BarChartApplet
 This java based bar chart applet enables you to add flexible and great looking bar and column charts to your web applications with just a few lines of html code.
 
BarCode
Add barcoding capability to your page or applets. This applet converts upper case letters and numbers into standard barcode. Source included.
Basic Calendar
Need to add a basic calendar to your site? This calendar is quick, clean and small.
BasicTOC2
BasicTOC2 is a simple Table of Contents JavaTM(JDK 1.0) freeware navigation applet with one-level folders and mouseover effects.
Bat
A one-sided Battleship game with source included. Try to find all the ships in less than 50 turns.
BattleShip
Try to sink the computer's ships
before he sinks yours in this
Java version of the classic BattleShip game!
Beam-Illuminated Text
You light up my life.... or maybe it's just my text. Either way, this laser-beam applet is a cool way to display your site's name! 
BFButton
An animated button with a background.
BillBoard
This cool little applet slides banner ads in from
random directions, even diagonal!
billsClock
An attractive analog clock with a moving second hand.
Biorythm Celebrities
Discover your ideal celebrity! The Biorythm Applets use the official calculations. Whether you believe in the theory or not, it is always fun to see. The results often seem to be right... 
Birdbrain
This applet is based on a classic game written long ago, on a platform not far away. The object is to move your player in a manner that causes the enemy to self destruct by colliding into each other.
BlackJack
One-player blackjack, versus the dealer.
BlackJack
A Black Jack Game with a nice layout.
Customizable foreground and background colors as well as the amount of
money the player starts with.
BlackViper
 
A nibbles style game with nice graphics and sound effects! 
Blink
Multi-colored blinking text. The included source is good learning tool for beginning programmers.
BlinkItem
An image with a blinking neon sign.
blinkmap
A visual site navigation tool.
Blox2
3D virtual legos you don't have to pick up when you're done.
BMI
Over-weight? Concerned about your health? Check your body mass
index (BMI) from the comfort of your own computer chair!
BMTron
A Tron game with hi-tech graphics for 2, 3 or 4 players; play against the computer or a friend. Avoid running into the wall, opponent's tail or yourself! 
BookFlip
Transition between images with this cool animated page flip applet! 
Bookmark Menu 
This applet reads a bookmark file (in Netscape format) and
displays the corresponding menu, like the one in Netscape's toolbar.
The bookmarks can be displayed in Internet Explorer also.
Bookshelf 
Cool little applet that presents your links as a bookshelf with
 each book representing a different link.
 
Boulderdash
Remember Boulderdash from the old Commodore 64 days? Here it is as
an applet, but without the scroll!
Grab the diamonds, but avoid the falling rocks and swarming fireflies!
Ten levels of excitement with full instructions on how to create
additional levels! Source included!
Bouncing Balls
You can move and resize the rectangle! Manipulate the bouncing balls.
Great for learning Java. Source code included!
BoomButton
An exploding button, with sound effects.
BooZerScroller
BooZerScroller is a horizontal scroller created by author Stefan Mateescu with some cute effects and loads of useful features.
BounceItem
Images bouncing around, with sound effects.
BreakOut
Interesting variation on the old breakout game. Source included!
Breakout
Breakout is an extremely addictive variation on the old breakout game. Add some fun to your site!
Brain Damage
A cool 3D kaleidoscope that has thousands of animated patterns
generated in real time!
bridges
Bridges is another free game you can add to your site. Can you make it to the other side before your computer does? Source is included!
BSCButtonBar
An easily customisable button bar. Source included!
BSCOutline
BSCOutline Java Outline is a customizable tree view control, similar to that used by Windows File Manager or Explorer.
BTJ
Back To Java is an opensource application which can generate Java Code from ByteCode.
Bubbler
Bubbler is a small applet which will make any image bubble when you
move your mouse over it or can be set to constantly bubble away on
your web page.
Bubbles
Slowly rising multi-colored bubbles.
Bumpy
This slick applet uses a "bump map" to create the effect of a spotlight passing over an image. Put your own unique logo in the spotlight!
Bumpz
You've seen some of the others, now check out Bumpz! Bumpz creates a spotlight type effect on your chosen image, and it's configurable!
	
	BurnShow
	Add this hot effect to your slideshow and watch your images melt from one to another! A code generator is provided for the HTML challenged! 
	
Bustout!
This free applet reminds me of some of the original "pong" video games. Break out all of the bricks to proceed to the next level.
ButtonBar
Buttonbar Applet lets you quickly add an attractive animated button bar
to your Web pages. Horizontal or Vertical display!
ButtonList
ButtonList is a	FREE button listing applet, used to navigate URLs.
