RadarDrome
A flexible applet for creating realistic radar-like effects with moving text. Free.
Radio
Radio is a slick little applet which enables visitors to your site to play six different .au files.
Rain!
The Applet that refuses to die. First snow, then confetti, now rain!
RainbowText
A simple, but nice effect: let your text cycle through all the colors in the spectrum. Source included.
RajServer
If you're looking for a 100% Java Web server, you'll want to check out RajServer, the free 100% Java Web server.
RatInMaze
The challenge of the free RatInMaze game is to help the rat out of the maze, of course. The rat is hungry and responds to the cheese you leave for it, so keep the rat alive by feeding him with the cheese as you lead him out of the maze.
RealChat
A chat software package that features colored text, pictures,
sounds, emotions, multiple rooms, private messages, user created rooms,
administrative manager, console, file logging, and more!
RealCheckers
Play this classic board game against the computer!
Try to jump all of the computer's pieces before it jumps yours.
Free for non-commercial use.
RealConverter
RealConverter converts an amount in one measurement of your own customizable set of measurements into another. Use it for currencies, distances, timespans, weights,... Colors, amount style, language, size and much more! 
RealInvaders
The earth is being invaded and you are the first line of defense ... the only defense!
Play this space invaders game and save the world - maybe the universe.
RealSnow
Generates real looking snow that falls over your picture. You can set a background picture
yourself and specify the amount of snow and the wind turbulence. IE only. Free.
Recover
Ah, those great slider puzzles. This one is fully configurable complete with sound!
redbugs
The FREE redbugs applet will keep you in action for hours!
RGB
RGB is a free applet that allows you to easily choose colors using scrollbars or a list and coverts between decimal and hexadecimal RGB values.
RiadaBillboard
This applet features fades, swipes, background colors, and a slick GUI interface for setting the parameters!
RiadaButton
RiadaButton is a graphical tool which enables non-programmers to create attractive button type effects for their Web sites.
RiadaForm
Using RiadaForm, anyone can quickly and easily design professional looking Web forms without any Java or CGI programming.
RiadaHeadline
RiadaHeadline is a highly configurable applet which scrolls headlines on your page.
Riada Melt
Melt away boring web sites with RiadaMelt! A slick GUI application does all
the work of dynamic text effects, transitions and colors, exporting everything
you need to the directory of your choice!
RKBlackJack
Most online card games eliminate a lot of the intricacies of the games they're based on. But RKBlackJack is a very complete blackjack game!
RMS_HR Divider Bars
A package of three small, animated divider applets! Awesome!
Rocks
The object of the game is get the Rocks to pass through the gates to the bottom. Each rock which makes it to the bottom will increase your score by one.
Rocks2
Rocks2 is a free sci-fi shoot 'em up that you can use on your Web site.
Rotate
A revolving image applet that does not need multiple images. Free.
Rotator
An extremely quick animator.
RotateInfo
RotateInfo implements a simple information rotator with basic
shading and shadow effects.
RTreed
Java applet that creates Trees and menus for your web site or java application.
It supports icons, colors, fonts, backgrounds, animations. Highly configurable
and there is a "source code license".
Rwho.java
Rwho reads the output of a CGI file, in this case "rwho," and displays the data within a scrolling "ticker." Great programming example!
Running Rainbow
Free, very configurable animated rainbow banner with source included.
Rush Hour
It's that time of day. Stuck in traffic again! Help the red VW get home. Free.
