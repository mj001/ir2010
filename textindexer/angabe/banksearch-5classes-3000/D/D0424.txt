IABanner
IABanner allows the user to page through a series of images and text blurbs. It includes a thorough configuration tool.
ibtn
A configurable animated button navigation bar with source code.
ICalendar
Time to start decorating for the holidays, and what better way than with
a time-released, interactive Christmas applet!
ICA Freeware Jigsaw
Enjoy a virtual jigsaw puzzle, or change the image to create your own!
ICrossword
Very slick crossword game with customizable sounds and colors.
iebb
Implement an IE style button bar on your page with this cool little applet!
iFrame
If you hate doing framed layouts, iFrame will get you out of all sorts of fixes.
Once in your page, you never have to touch your layout again -- because
iFrame gets its content from a separate text file.
igneus
The Igneus Java Menu System is a way for web developers to create powerful and dramatic hierarchical menus within web sites.
Infinite Icon Menus
The Infinite Icon Menus applet is one of the slickest menu applets we've seen! It's a part of the Web Effects collection available from OpenCube.
ili
ili permits web site designers to add an ASCII art editor applet to their sites, or it can be used with an applet viewer as a stand alone ASCII art editor.
ImageAudio
When you pass your mouse over the applet, you'll see a highlighted area and hear an audio clip. Source is included, making this a great tool for learning Java.
ImageFader
Create your own fade transition with this cool little applet! (The secret's in the fade.gif).
ImageMap
An interactive image map with sound effects.
ImageTape
Scrolls multiple images horizontally.
ImageToday
This shareware applet allows you to rotate images by week, day, hour, or minute.
Each image can link to a different URL.
ImageViewer
Compact method for displaying your images. Click the thumbnail to see a full size version!
Image Viewer
Awesome image viewer with thumbnail and full size mode. Easy to install with configurable skin color! 
ImageViewer
Zoom, Zoom, Zoom! That's what you'll be saying when you see this customizable grid image viewer that zooms thumbnails to full size with the click of a button. Supports .jpg files.
ImageViewer
This small applet allows you to organize you pictures into groups and cycle
through them with pull down menus and buttons. All the information about the
pictures is located in an external text file, so updating the applet is easy!
IMap
How many times has someone requested pop-up help windows on imagemaps? Here's an elegant solution.
ImageZoom
Take a closer look at any image/photo on your Web site. Fully configurable, and FREE!
ImageZoom2
An advanced version of ImageZoom, this applet allows gradual zoom and pan and JavaScript control.
ImgShow
Watch your images slide across the screen with this free family slide show applet.
iMMbar
This is an imagemap menu for the top of your webpage. Create your own image for the applet, and then it can attach drop-down bars to your hotspots on the imagemap. Size, text and colors are all configurable.
Impression
Viewer creates an impressionistic painting of an image.
Inferno
Make your message sizzle! This applet will light your fire.
Try moving the mouse over the applet when text is being displayed.
IntelliJ Renamer
IntelliJ Renamer is a multi-platform Java source code management tool.
Analyze dependencies, find usages, rename and move packages, classes, methods and class variables.
InternetTime
Internet Time is a decimal time system invented at Swatch and MIT. It divides the day into 1,000 beats, each equivalent to about 86.4 seconds (0:01:26). Time zones are ignored, so @715 is the same time in Sydney as in London.
Intro 1.1
Intro 1.1 morphs any number of images into each other. It comes with lots of parameters to play with! 
invaders
Nostalgic shoot-them-up applet for those who used the play the game on C64 (and even before that, pehaps on a VIC20).
iPOP
The iPOP applet emulates the Windows start button, and offers multiple pop-up menus which can overlap any page content, including frame edges.
iPopper
iPopper is a cool little toolbox applet that allows you to place a
system-like Popup Menu anywhere at all on the page. iPopper pops up
anywhere, even outside the applet and supports up to 100 menu items!
IQGame
A slick, challenging, free game based on an old classic. Try to leave either as few or as many marbles as you can.
IQ Quiz
Test your IQ! A cool intelligence test that returns your
IQ score. Source included!
Iridium
Save the planet Irue from destruction in this 10 level
action game.
Iruvar
This applet reads a file which contains two lists of items which have one to one relationships. Example - States and Capitols.
ISCBanner
ISCBanner V1.2 is a Java image banner which allows images to be scrolled to the left, right, down or up, or simply acts like an image animator.
ISCBlinkButton
ISCBlinkButton V1.0 is a navigation button applet that lets the user define font size, text color, blinker color, button alignment, link URL, background color, etc.
iScrollKitFree
iScroll is a quick, easy way to place a "marquee" text scroller on your page. Highly configurable.
ISCUpScroll
ISCUpScroll V1.2 is a configurable text display applet that allows users to define their own font size, font color, text speed and step, etc.
iSlide
An incredibly rich suite of navigation applets. We can't even begin to show everything that iSlide can do for you.
iTab Menu Systems
Free java tab bar applet which drives layers to provide a fast flick-through navigation to the main sub-sections of your website. 
Optionally adopts system colours of client machine to blend in with the user's browser bars. 
iTree Menu
Formerly known as Alan's Tree Menu, this is a free applet in which the programmers have placed emphasis on compactness, ease of configuration, smoothness and speed of operation.
