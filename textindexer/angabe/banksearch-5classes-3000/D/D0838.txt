Calc
The Calc applet is an actual calculator with advanced functions including symbolic differentiation.
Canadian Mortgage Calculator
A calculator designed for fixed Canadian Mortgages. It allows "what if" analysis to determine the best mortgage to suite your budget.
CarGame
CarGame is a free racing car applet developed by author bharath for fun. Source code is included, so you can learn while you play!
centipedo
Rid your garden of pests in this classic video game.
CGIFormParser
This class URL decodes and works with a string resulting from a CGI input form.
Chart
A parameter-based bar chart. Parameters can be set dynamically from a cgi or JavaScript. Source included!
Choose your Calendar
This applet displays the calendar of any month and year combination.
Find out the day of the week you were born! It's very easy!
CircleBanner
If you're looking for a professional looking effect for your site, CircleBanner may be just the applet. You can change the parameters for the font, color, direction, size, background image and more on this free applet.
ClipControl
Play or loop an .au file in a web page, with optional controls.
Cliffie
Quite a programming feat, and a real showcase of using sprites, Cliffie is one of the slickest games on the JavaBoutique.
Cline
Cline is another free button applet. The surface of the button is algorimetically animated, and has several moving colored lines on it.
Clock
The original analogue clock applet from Sun Microsystems. Source is included.
Clock2
A clock that can be configured in a variety of ways.
ColorFinder
This applet lets you use scrollbars to modify the Red, Green and Blue colors in the applet, instantly showing the hexadecimal value of the new color.
Colormenu
A Colorful menu applet with mouseover effect. Highly configurable.
Comes with commented source code, great for learning Java!
colors
Displays colors for user-defined RGB hexadecimal combinations.
Confetti
Starting life as snow, this trendy applet was converted to confetti by popular
request! Highly configurable, including various amounts of up to 13 different
colors, pixel size, delay, fits any size image and supports frame overlay!
Connect4
Just like you remember from when you were a kid, but the computer opponent is tougher than your friends were! Source included.
CoolScroll
CoolScroll features smooth scrolling text (using your own fonts!) which bounces up and down and fades in and out!
Corf_Scroller
This is a small horizontal text scroller applet (7k) with source code. It is unique in that it sets font size automatically. Highly configurable with url launch included.
Cornholio
Let users call up sound files with this basic audio applet. Source is included, and this is a good example if you want to begin coding sound in your Java projects.
countdown
Counts down to a user-defined date and time.
CountDown2
This freeware applet displays the number of days till any specified date. You can use your own digit images, define a background image or color and set the number of pixels between the digits. Source included!
Cross Fader
This little applet produces two lines of text that fade in/out over top of each other. Source included!
crossword
A 15x15 crossword puzzle.
CubeApp
Rotate the cube by dragging it with the mouse button; move the pieces by holding "control" and dragging one piece to a neighboring piece with the mouse button. The Rubik's Cube of the Web.
Cursor
Make some cool 3D effects with this applet. Every time you click a new line is added!
