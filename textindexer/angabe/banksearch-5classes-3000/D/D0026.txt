SameGame
Click two or more blocks of the same type, that touch horizontally or vertically to make them disappear, but pay attention! The more blocks in a group, the higher your score when you click! Source included!
SamosMenu
A fast, free, attractive menu applet.
SayItNow!
Speech enable your website with SPEECH SYNTHESIS; the applet plays back any phrase you type! This is not steaming audio!
Scoreboard
It's almost like being at the game! Here's a configurable text animator from John Morris of the JavaPlayground.
ScreenRes
ScreenRes detects the viewer's screen resolution. This can be taken a step further to create pages which look good regardless of the user's resolution.
Scroll
Very configurable typewriter scroller with source included!
Scroll3D
You've probably seen many text scrollers, but you probably haven't seen many 3D text scrollers like this one.
Scroller
A very nice, small, simple text scroller with shadows. Free.
Scrolling Text Applet
Provide an animated look and feel to your web pages with Scrolling Text. 
ScrollerII
A highly configurable horizontal text scroller that can scroll HTML from
a text file!
Scroller Component
Scroller Component can be used to display vertical or horizontal scrolling text with hyperlinks. It can be used to scroll text/image combinations and play an .au file on mouseover. Still not satisfied? Then use it for panoramic views! All in one applet! Highly configurable.
ScrollIt
Horizontal scrolling text applet with configurable
text color (including link and hover color), font style
and size, and target frame for linked page to open in. Comes
complete with stop, rewind and forward buttons! Cool!
ScrollItUp
A highly configurable vertical scroller with it's own rewind/forward/stop buttons created with drawing routines within the Java code!
ScrollText
A small (4kb) horizontal scroller that scrolls both ways. Very easy to configure and free!
Scroll Window
Scroll Window is a free java applet that can be used to add a scrolling text window to your webpage. Source included!
ScrollWriter
ScrollWriter combines the typing effect of our TypeWriter applet with the same
smooth scroll of our popular Vertical Text Scroller. This applet's unique effect is
sure you make your web site stand out from all the rest, while maintaining a
professional appearance.
sdmenu
This is a slick configurable menu which uses a background image.
Search
This Java application runs on your desktop, retrieving HTML files from the web without using a web browser.
Search2
Give your visitors the option of searching... one, many, or all of the 
major search engines! Source included!
Search Me!
Search Me! is a client side search facility that can be added to Web pages.
It searches Web pages set by the webmaster for keywords required by the
user. As it requires no server side cgi scripts, it is ideal for sites built on
a budget whose Internet service providers do not allow server side processing.
SearchApp
SearchApp is a free tool that your visitors can use to search the Web using the major search engines.
Searchlight
Nifty image-fx applet - A searchlight follows the mouse and moves around randomly when the mouse is not over the image.
SearchToHTML
The SearchToHTML applet performs a simple text search of the files you specify in its parameters. It writes out the results of your searches as HTML in separate frame or a new window.
SecretPass
Password protection without static HTML or a text file! The applet
redirects to a file of the same name as the password. The password
itself is not stored anywhere - leaving hackers with nothing to
find!
Securer
Securer applet gives people without cgi access the option of adding password protection to specified directories of their website.
Sensomap Mini
 
Sensomap displays an image file (such as JPG or GIF) and turns it into an imagemap. It can handle both polygonal and discontinuous shapes, making it very useful for geographical maps and complex technical diagrams. Preloaded highlight images allow fast emphasis of hotspots. Context-sensitive textual information can be defined by the user to appear in a display panel at the side of the map. 
shadowText
Cut your load time; add shadows to text without graphics!
Background, text and shadow colors are customizable.
ShapesApplet
A simple applet to illustrate multithreaded programming. It consists of shapes, each running in its own thread that ebb and flow at different rates.
SheikLines
The SheikLines applet is similar to the famous Windows screen saver, except that in this version text fades in and out along with the lines.
shifter
Slick image effect applet; slices up images and scrolls the segments in from both sides of the screen.  Configure with as many images and slices as you wish.
Shimmer
Add a cool shimmer effect to your gif images with this little (4.04kb) applet!
Shooter
The creator of Sokoban and Eliminator has produced yet another game to occupy you
for hours! Destroy all of the duplicate bombs in a set number of moves. Source included!
ShootingGallery
Now in an updated version with source code included, ShootingGallery is a simple but fun game, with nice graphics/animation. Shoot the ducks (they speed up!) to get points and knock them over.
ShowColors
This simple applet is a tool to quickly choose colors for java R&D. You can type in RGB values, darken or lighten them, or choose from preset colors.
Siji
An old fashioned peg game with in a new dimension. Remove one marble at a time until only one is left. Forty-five levels and source is included!
Silly Scroll
Silly Scroll applet features 5 different text effects. Scroll your
text smoothly from left to right, right to left, or vertically. You
can even have your text fly in from the bottom, or grow one letter
at a time!
Simon
This is the classic Simon game in Java. The game gives you a sequence of
colors to repeat back to it. Try to beat your own high score! Free.
SimplePlot
Plots simple two-dimensional functions according to user specifications. Source included.
SimplePong
This applet is configurable in that you can change the colours, the background image and the difficulty of the game.
simpletext
Quickly displays a text file in a static area.
SineText
Scrolls text along a sine curve.
SinuzLogo
SinuzLogo is a slick little applet which creates a rippling type of effect on the image which is displayed.
SIR 3D Charts & Graphs Collection
Five, count 'em, five! of the latest feature packed, highly customizable graphing and charting applets with documentation and examples! Source code is also available.
SIR Advanced Tree Explorer
 
Highly customizable tree applet with mouse over descriptions of each link! 
SIRfadeFree
This applet cycles through images and fades the image in and out between images. This free version contains most features.
SIRFreeTree
This free java applet provides a sophisticated Tree style view to your web pages.
SIRhmenuPro
A professional navigation system that's easy to implement. Highly
customizable; everything is adjusted
by parameters in the HTML tags.
SirTypeFree
Easily implement a high impact text display with this free text animation applet.
SlickTree
It's all in the name. A tree applet with a really slick mouse over.
Vertical Menu System
A professional two-level, vertical, navigation system.
SiteAnim
This free applet allows visitors to your site to create animated Javascript-based banners and images. The applet writes the HTML and JavaScript code for you!
SiteBrowser
A highly customizable navigation applet with seven, (yes seven)
different layout variations!
SiteRemote
SiteRemote is a fully customizable 3-in-1 applet consisting of a PhotoViewer, Remote and Audio Player.
SiteSearcher
The FREE SiteMapper applet performs a simple search of the set of files you define.
SiteSearch
This free applet searches a Web site for keywords by parsing and following links on the site.
SiteSticky
SiteSticky is a Client/Server system (in Java) for enabling Web based chat,
entertainment and customer service. Includes text and voice communication, a
shared white board, multi-user games, co-browsing, an open API, and much
more !
Skeet
Test your skill with this new action game. Shoot the skeet and the squirrel.
SkyLogo
This is a slick effect which combines a moving background image with color-changing text!
Slate
A fun little drawing applet that helps kids practice their mouse-handwriting.
Slide Servlet
A Java servlet that allows you to share documents over the web.
Slider
The free Slider applet functions much like a banner, and is highly configureable. This is a great way to direct attention to your latest news, links, headlines, etc.
slidepuzzle
This free applet is similar to those slide puzzles you played with as a child on a road trip. Move the pieces around by clicking on them with your mouse.
SlideShow
This applet cycles through an unlimited number of images waving them in and
out of focus.
Slideshow
This applet presents a slideshow, complete with descriptions for each slide. Very slick interface, perfect for presentations.
SlideShowApplet
Want to present your site to visitors as a slideshow? SlideShowApplet allows you to present one page after another, creating a Web presentation!
SlideWays
A smooth sliding banner applet! Can be used horizontal or vertical.
Images can be different sizes and each image can have a link.
Includes many different parameters that can be changed.
SlideViewer
SlideViewer is a free applet that is used to display a sequence of GIF or JPEG images with a caption. The applet provides random transitions between the images much like MS PowerPoint.
Sliding Text
Let your text sliiiiidddde into place.
Sliding Tree Menu
Tree Style navigation menu with unlimited sub menus and JavaScript
interaction. Configure fonts, images, associated icons, and more!
Slipper
CodeBrain's Slipper is a highly configurable slide-show applet. It
supports up to 100 images of the same or differing sizes all of which
can be individually linked! Use it for a navigation applet and keep
your visitors hopping!
SlotMachine
If you're looking to add some pizzazz to your Web site, the SlotMachine applet may be just the ticket! With sound effects and your own prizes!
SmartIcons
SmartIcons is a free button applet which enables you to create a slick navigation tool for your site. Create your own tool tips!
SmartTree Menu
Find out why SmartTree Menu is billed as "the world's first Java navigation application that accommodates Automatic Relative Opening."
Smif
Display your image or banner with this cool animated noise effect. Highly configurable. 
Snake
Another version of a popular game. See how many blocks the worm can eat
but don't run into the walls or yourself! Source included!
SnakePit
The free SnakePit applet is a variation of a simple yet addictive game that has been ported to countless platforms.
Snow
Cool off with this slick little snow effect applet.
SnowDrift
Decorate for the holidays with this snow applet that features
your own personalized message scrolling serenely across the
image!
SnowFlakes
Not your normal snow applet... this one has real flakes plus a mousetrail! 
SnowShow
It's no show..... er snow so...... aw heck, check out this cool slide show with it's very own snow effect!
SnowMenu
SnowMenu lets you create a menu with a snowing background image. It is simple yet flexible. Free
SolidScroller
A horizontal scroller that will jump or wave your text across the screen.
A cool mirror effect can be added to the bottom! 
Sonali Calendar
 
A handy, customizable calendar, suitable for printing! Add your own background image.
solitaire
A quick puzzle: Remove all 44 marbles from the board. Source included.
Souvenir
Souvenir reads a list of images from a text file and displays the images by fading them in and out. Free.
SpaceFlight
The free StarFlight applet is one of the first applets that simulates a real flight through a starfield!
StarGaze
Your image, overlayed with a flowing starfield, topped off with a vertical scroller! What more could you want? 
SpaceLinks
SpaceLinks is a FREE link menu applet with an animated "flying through space" background.
SpaceMaze
Raid Alien Space Stations of diamonds they have stolen from Earth!
SpaceWar
A free, classic Galaga-like shooter. The aim is to destroy 20 of the evil RichbaStards control satelites, save the world, and win the girl.
SpaceWave
You're traveling through space when all of a sudden, there's some words coming at you, waving like a snake! Cool applet!
Sphere of Chaos
Spheres Of Chaos is author Alexander Zbiek's own version of the famous asteroids applet! If you can play it without getting addicted, try your luck!
Spice2
A cool little navigation applet that provides
a dropdown menu inside of a scrolling image.
SpinTop
An animation applet that can be reversed by a mouse click.
Spiro
Draws spirographs according to various user-defined parameters.
SPlane Bean Applet
Displays a real time body plot and time response for poles and zeros
in an S-Plane. Drag poles/zeros to change display.
Splash
A fun little wave/ripple simulation with source.
SpotLight
This cool little applet simulates an interactive spotlight with soft
shadowed edges.
Spotlight
The free Spotlight applet creates a spotlight effect over your image! Light up your Web site with this slick visual effect!
SpreadSheet
A simple spreadsheet, with addition and multiplication.
Sprites
This cool little applet animates multiple sprites in a scene.
SQLFun
SQLFun is a Java program/applet that allows you to experiment with SQL statements without having to install an SQL server or database program. All you need is Java and ODBC or any other database that supports JDBC.
StarBase
Yet another shoot 'em up video game.
stardate
The stardate applet displays the current stardate -- both the human and klingon versions. Java 1.1 is needed to run (NS4 or MSIE4). Source included.
Stardust
Picture a field of stars rushing towards you, picture text appearing
out of the void... Now picture it on your website! A cool applet
with source included!
StarField
A scrolling starfield simulation.
starfield 1.0
The free starfield 1.0 applet allows you to scroll text over a starfield background! If you've been looking for an applet that doesn't look like all the rest, starfield 1.0 may be perfect for your needs.
Starwars Text Applet
In a galaxy far far away, your message appears... scrolling through a 3d starfield.
Starfield Simulation
Put your name - or anything else - in outer space. This applet will make you think you are flying high! Free.
StarfieldUpscroller
This is a free scroller applet which shows a 3D starfield in the background. This applet includes some useful features that are configurable!
StarFlite
This is a Java applet which creates 3-D star shower on your Web page. You can change the size and shape of the applet to suit your tastes.
StarGame
This is an arcade game in space! Included with the applet is the source code fully commented! Also included is a template for game applets! This is a must for all those interested in java game programming.
StarGate
Add style and class, (as well as animation) to the way your visitors send mail from your site! Add your own touch, skin change is supported!
StarLink
Part of the Multi-Board applet package. This novel navigation tool will certainly get noticed! Highlight one of the three brighter stars to see its information.
Stars
The stars glittering and a full moon - what could be more beautiful!
Use your own landscape and moon. Free
StarSim
Looking for a slick space simulation applet? StarSim loads fast, uses double buffering to eliminate flickering!
StarsMenu
A fully customizable animated text menu which is flying in space.
StarUp
This is a free upscroller applet with a 3D starfield in the background. You can alter the parameters to customize the applet.
StarWarsScroller
This applet is frequently requested! This is the effect you all know from the Star Wars intro. Now you can feature a Star Wars type effect on your site! Free for private use!
StarWords
The StarWords applet is designed to be used as an eye-catching and original heading for use on your Web pages.  User defined messages fly towards the viewer atop a rotating 3D starfield.
StockChart
StockChart displays stock value evolution in two modes: The
standard display and the The Japanese Candle Chart display.
Street Hockey
 
This popular game was one of the first computer games ever. Test your skills and see if you can beat the computer! Source included! 
StringGradient
StringGradient is a class that will turn any String into an image where the
string is displayed with a gradient. This is a tool for java programmers who
want to add some fancy graphics to their applets.
StringWalk
A sweet little (5kb) news/link ticker.
Stroids
Stroids is an exciting new Java game based on the GameCore Java programming library. Its 3D raytraced graphics and smooth game play set it apart from other Asteroids clones on the net.
Special User Interface
Due to the lack of decent GUI components in the JDK 1.1, it's not that easy to generate a nice user interface, and then came SUI to save the day!
SunRise
Enjoy the sunrise along with your text string! This applet is entirely self contained in one class file, making a fast download!
Swarm
This appet demonstrates a simple flocking, or swarming behavior. Each particle follows three simple rules: follow the leader, attempt to match neighbors' velocity and stay near the center of the swarm's mass. If you move the mouse over the applet, the mouse becomes the leader, otherwise the green particle is the leader and will attempt an avoidance behavior. Source included! 
SWServer
SWServer is a simple free Web server Java application, with options to simulate modem transfer and resumable file loading.
