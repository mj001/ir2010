Please use this visual calculator to 
compute the value of any one of the 16+ million colors in the 24 bit RGB 
(Red-Green-Blue) spectrum. You can then assign this color as the background or 
foreground color of any of my applets. 
 
 
To use this tool, type in your red, green, and blue value, click on the Calculate Button, 
and the tool will calculate your RGB value and display the color. 
 
This tool requires a JavaScript and Java-capable browser. I have tested it with Microsoft 
Internet Explorer 4.01 for Windows 95, Netscape 3.01 for Windows 95, and 
Netscape 4.04 for Windows 95.
